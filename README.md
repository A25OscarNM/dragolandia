# Dragolandia

---

Breve explicacion de que va el codigo, que estamos haciendo y que es lo siguiente que vamos a hacer

En este proyecto estamos creando una aplicación con interfaz gráfica con temática medieval utilizando MVC, Singleton, hibernate.
Tendrémos a un mago al cual podemos controlar que hechizo hace y monstruos que se enfrentaran a el intentado matarlo.

Actualmente estamos cambiando la parte de hibernate para que utilice Singleton y los próximos cambios son utilizar Joins para un par de consultas del controllador para obtener los monstruos que estan disponibles para ser añadidos.
